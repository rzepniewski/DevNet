What is Machine Learning?

"A computer program is said to learn from experience E with respect to some class of tasks T and performance measure P, if its performance at tasks in T, as measured by P, improves with experience E."
Tom Mitchell Machine Learning, 1997

Traditional Programming

  • Human writes explicit rules
  • Rules + Data → Output
  • Brittle to edge cases
  • Requires domain expertise

Machine Learning

  • Computer discovers patterns
  • Data + Output → Rules
  • Adapts to new patterns
  • Requires labeled data

Key insight: Instead of programming rules explicitly, we provide examples and let the algorithm discover the underlying patterns.
