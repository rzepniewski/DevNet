    Multi-VM Environment Example¶

    Deploy a complete multi-tier application stack with web, application, and database servers.

    Overview¶

    This example demonstrates a typical 3-tier architecture with:

    • 1 Web server (frontend)
    • 1 Application server (backend)
    • 1 Database server (data tier)

    Architecture¶

    graph TB
    subgraph "3-Tier Application Stack"

        A[App Server<br/>192.168.1.11]
        D[DB Server<br/>192.168.1.12]
    end
