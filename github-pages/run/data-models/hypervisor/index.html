    Hypervisor¶

    Data model key: hypervisor

    Defines where VMs will be deployed. Supports multiple platform types with vCenter and standalone ESXi currently implemented.

    Structure¶

    hypervisor:
  vcenter: [...]              # vCenter Server deployments
  standalone_esxi: [...]      # Direct ESXi deployments
  openstack: {}               # Future: OpenStack cloud
  proxmox: {}                 # Future: Proxmox VE
  nutanix: {}                 # Future: Nutanix AHV

    vCenter Fields¶

    Field Type Required Default Description
    host string Yes - vCenter hostname or IP address
    port integer No 443 vCenter port
    username string Yes - Administrative username
    password string Yes - Password (use Ansible Vault encryption)
    validate_certs boolean Yes - SSL certificate validation
    datacenters list No - List of datacenter configurations
    ova_import_async_timeout integer No 3600 Max seconds to wait for OVA import (min: 60)
    ova_import_poll_interval integer No 45 Seconds between async task status polls (min: 5)

    Datacenter Fields¶

    Field Type Required Default Description
    name string Yes - vCenter datacenter name
    clusters list No - List of cluster configurations
    non-cluster object No - Non-clustered ESXi hosts in datacenter

    Cluster Fields¶

    Field Type Required Default Description
    name string Yes - vCenter cluster name
    datastore string No - Default datastore for cluster
    vm_names list No - VMs to deploy to this cluster
    esxi_hosts list No - Specific ESXi hosts within cluster

    Non-Cluster ESXi Host Fields¶

    Field Type Required Default Description
    name string Yes - ESXi host FQDN or IP as registered in vCenter
    datastore string Yes - Datastore name on this host for VM storage
    vm_names list No - VMs to deploy to this specific host

    Standalone ESXi Fields¶

    Field Type Required Default Description
    name string No - Optional logical name for this ESXi host
    host string Yes - ESXi hostname or IP address
    port integer No 443 ESXi port
    username string Yes - Root or admin username
    password string Yes - Password (can be plain text or Vault encrypted)
    validate_certs boolean No false Whether to validate the ESXi TLS certificate (lab self-signed certs)
    datastore string Yes - Datastore name for VM deployment
    vm_names list No - VMs to deploy to this host
    ova_import_async_timeout integer No 3600 Max seconds to wait for OVA import (min: 60)
    ova_import_poll_interval integer No 45 Seconds between async task status polls (min: 5)
