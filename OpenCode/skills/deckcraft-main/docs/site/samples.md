## About the Samples

These presentations were generated with DeckCraft and demonstrate:

- **Glassmorphism design** — frosted-glass cards, animated mesh backgrounds, gradient accents
- **Multiple themes** — mesh, cyan, emerald, orange, cisco, dark, and more
- **Rich components** — stats, timelines, charts, code blocks, diagrams, math equations
- **Fragment animations** — incremental reveal effects per slide
- **Self-contained output** — every file embeds all CSS and JS; share by email, Slack, or host anywhere
