Types of Machine Learning

ML algorithms are categorized by the type of feedback available during training.

Supervised Learning

Learning from labeled examples where the correct answer is known.

  • Classification: Predict discrete categories
  • Regression: Predict continuous values

Example: Spam detection, house price prediction

Unsupervised Learning

Finding hidden patterns in unlabeled data without explicit guidance.

  • Clustering: Group similar items
  • Dimensionality reduction: Compress features

Example: Customer segmentation, anomaly detection

Reinforcement Learning

Learning through trial and error with reward signals.

  • Agent: Makes decisions
  • Environment: Provides feedback

Example: Game playing, robotics
